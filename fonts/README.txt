SV 名片模板字體包
===================

用途：在新機器或缺字的機器上，一次裝齊所有名片模板會用到的字體。
裝好後 Illustrator 開模板就不會跳「遺失字體」，文字也才編輯得動。

────────────────────────────────────────
一、怎麼安裝（二選一）
────────────────────────────────────────

【最簡單】
  雙擊本資料夾內的「安裝字體.command」→ 會自動把字體複製到
  ~/Library/Fonts/ → 完成後請「完全關閉並重開 Illustrator」(⌘Q)。

【手動】
  選取本資料夾所有 .ttf / .otf → 雙擊 → 字體簿(Font Book)按「安裝」
  → 完成後完全關閉並重開 Illustrator。

※ 重點：Illustrator 只在「啟動時」掃描字體，新裝字體一定要重開才生效。
